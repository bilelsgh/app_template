<b>This app allows you to signup and login with Angular, Express and MongoDB.
Before starting you have to install some modules. You can use it as basis for your project.</b>

- In /frontend : 
  - npm install -g npm@latest
  - npm install -g @angular/cli
  - Install bootstrap and add it in angular.json
  
- In /backend : 
  - npm install -g npm@latest
  - npm install -g @angular/cli
  - bcrypt
  - jsonwebtoken
  - mangoose
  - mangoose-unique-validator
  - bodyParser
